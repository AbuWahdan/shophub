
### Important Files
| File | Purpose | Size |
|------|---------|------|
| `lib/main.dart` | App entry point | 100 lines |
| `lib/src/model/data.dart` | Mock data (50 products, orders, addresses) | 400+ lines |
| `lib/src/pages/product_details_new.dart` | Full product page | 400+ lines |
| `lib/src/pages/search_filter_page.dart` | Search & filtering | 350+ lines |
| `lib/src/themes/theme.dart` | Material 3 theme | 150+ lines |
| `lib/src/config/image_constants.dart` | Asset paths | 30 lines |

### Before You Start
1. Make sure Flutter 3.10+ is installed

3. Ensure you have 1GB+ free disk space
